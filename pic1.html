Mazda Skyactiv

Mazda Skyactive is a new type of chassis that Mazda engineered. It was designed to be stronger, lighter, safer and promote fuel efficiency. We put together a course that trained technicians and sales staff the newest features of the Skyactiv Chassis. We used 3D elements and video to make it as realistic as possible. This course was programmed using HMTL, CSS and Javascript.
